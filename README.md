## Tools

## Project Overview
Interactive healthcare analytics dashboard analyzing hospital and ICU occupancy trends across Australia (2020-2023).

## Features
- KPI monitoring
- Hospital vs ICU trend analysis
- Monthly and yearly pressure analysis
- Peak crisis detection
- Interactive slicers
